# Product Sense

## What This Is

Free, professional-grade trading risk management calculators for crypto, forex, and stock traders. The product is a static web tool — no accounts, no subscriptions, no data collection beyond analytics.

**Brand**: Risk Terminal
**Domain**: whatsriskmanagement.com

## Core Value

Traders need to calculate position size before every trade. Most do this manually or use clunky spreadsheets. Risk Terminal provides instant, accurate calculations with a terminal aesthetic that signals precision and professionalism.

## Target Users

- Crypto traders using leveraged perpetuals (Hyperliquid, Binance, Bybit)
- Forex and stock traders who need position sizing
- Beginners learning risk management concepts via the guides

## Product Principles

1. **Free forever**: No paywalls, no premium tiers, no account creation
2. **Precision over polish**: Terminal aesthetic communicates accuracy and seriousness
3. **Math correctness**: Leverage math must be right. Position size = risk amount / risk per unit. Leverage only affects margin.
4. **No backend dependency**: Everything runs client-side. The site works offline after initial load (except live prices)
5. **Trust through transparency**: Open source, no hidden tracking beyond GA

## What This Is NOT

- Not a trading platform or exchange
- Not financial advice
- Not a portfolio tracker
- Not a signal service or copy trading tool

## Features

### Position Size Calculator
Input: trade type, entry price, stop loss, leverage, risk amount
Output: position size (units), margin required, potential loss, risk percentage

### Profit Calculator
Input: trade type, entry price, stop loss, leverage, position size, up to 4 take-profit levels
Output: profit at each TP, total profit, average profit, ROI, primary R:R, average R:R, margin, potential loss

### Live Price Ticker
Real-time crypto prices from Hyperliquid WebSocket. Scrolling marquee with 24h change percentages.

### Educational Guides
SEO content teaching position sizing, risk-reward ratios, and trading strategies. Each links back to the calculators.
